# 🍔 Food Delivery Website

A modern, responsive food delivery web application built with **React 19**, **Redux Toolkit**, and **Tailwind CSS**. Features a beautiful UI with dark mode, smooth animations, and a fully functional cart system.

## ✨ Features

- **Modern UI/UX** with Light & Dark Mode
- **Real-time Search** with instant filtering
- **Category-based Filtering** (All, Breakfast, Soups, Pasta, etc.)
- **Fully Functional Cart** with quantity management
- **Persistent Cart** (saved in LocalStorage)
- **Smooth Animations** powered by Framer Motion
- **Toast Notifications** for better user feedback
- **Responsive Design** (Mobile-first)
- **Professional Cart Sidebar** with slide animations

## 🛠️ Tech Stack

- **Frontend**: React 19 + Vite
- **State Management**: Redux Toolkit
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Notifications**: React Toastify
- **Icons**: React Icons

📱 How to Use

Browse: Explore food items by categories
Search: Use the search bar for real-time filtering
Add to Cart: Click "Add to Cart" on any dish
Manage Cart: Open cart sidebar to adjust quantity or remove items
Toggle Theme: Use the sun/moon icon in the navbar
Dark Mode: Automatically saves preference
